# 提交媒体动图作业

调用SubmitDynamicImageJob提交媒体动图作业，开始异步处理。

**说明：**

-   只有状态为**上传完成**、**转码中**、**正常**、**审核中**和**屏蔽**的视频才能发起动图作业。
-   动图生产计入视频转码费用，且计费标准一致，按照分辨率和时长计费。
