# 提交媒体转码作业

调用SubmitTranscodeJobs提交媒体转码作业，开始异步转码处理。

**说明：**

-   只有状态为上传完成、正常和审核中的视频才能发起转码。
-   如需要获取转码结果，可接收回调消息：[单个清晰度转码完成](~~55636~~)、 [全部转码完成](~~55638~~)。
-   该接口可支持HLS自适应码流打包任务中对字幕地址进行动态替换，如打包任务无字幕打包处理，不建议采用该接口发起任务处理，而建议视频上传时设置对应的打包模板组ID即可自动发起打包流程。
